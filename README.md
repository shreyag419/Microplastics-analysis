Microplastics Detection & Analysis using Data Analytics

 Project Overview

This project focuses on detecting and analyzing **microplastic contamination in air and soil** using **data analytics, machine learning, and geospatial techniques**.

We aim to identify **high-risk pollution zones**, understand patterns, and create **visual risk maps** to support environmental monitoring and decision-making.


## Objectives

* Analyze environmental datasets (air, soil, land-use)
* Predict microplastic contamination levels
* Identify pollution hotspots using clustering
* Generate GIS-based risk maps
* Study impact of human activities on contamination

---

##  Tech Stack

* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib, Seaborn
* **Geospatial:** GeoPandas, PyKrige
* **Tools:** QGIS / ArcGIS

5. Machine Learning (Regression & Clustering)
